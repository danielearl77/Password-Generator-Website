Passwords and PINs

It is vital that for each account you have a different password is used. Using the same password on more than one email or social media site is asking for trouble, should one account be compromised if your password is the same then all other accounts could be compromised. Strong, random, and long passwords also go some way to slowing down an attacker. Avoid keeping passwords in an unsecured location, if you have a lot of passwords then consider using a secure password store application or service.

To generate a totally random password made up of a string or letters, numbers, or symbols use the Password Generator above, selecting the relevant options. The 'Remove Ambiguous Characters' will generate a password without 1 0 o l i L I as when read back later it can be unclear which character it is meant to be.

To generate a password made up of random words, which might be easier to remember, use the Password Phrase Generator above, selecting the number of words and whether you want random numbers on the end of the password.

To generate a new PIN then use the PIN Generator above, selecting the length of PIN required.

File Integrity

If you regularly upload or download files on the internet, then File Integrity should be important to you. If you download files from a website how can you be sure the file hasn't been tampered with to contain malicious code? If you upload files for other people to download how can you help your users be sure that the file they are downloading is the same untampered version you uploaded?

Generating a file Checksum and displaying it on the site with the uploaded file allows users to check after downloading that the file is the same. After downloading a file, as a user, you can generate a file checksum on your copy of the file and compare it with the one posted on the website.

MD5 File Checksum

Windows-based PCs

If you are using Windows as your operating system and need to generate a MD5 File Checksum then you can download Microsoft File Checksum Integrity Verifier here for free.

Mac or Unix-based PCs

If you are using a Mac operating system then you can generate MD5 file checksum using the built in Terminal application. Before you start make sure the file you want to check is in your Mac Downloads folder. Open 'Terminal' and at the prompt type in cd Downloads and press enter. Then type in md5 your_filename.here and press enter. The result will the be displayed. Caution must be exercised when using the Terminal on a Mac. When you are done type exit and press enter, you can now close the Terminal application.

UNIX-based systems will vary but you should be able to use the MD5 command from the prompt to generate the checksum.

SHA1 File Checksum

Windows-based PCs

If you are using Windows as your operating system and need to generate a SHA1 File Checksum then you can download Microsoft File Checksum Integrity Verifier here for free.

Mac or Unix-based PCs

If you are using a Mac operating system then you can generate SHA1 file checksum using the built in Terminal application. Before you start make sure the file you want to check is in your Mac Downloads folder. Open 'Terminal' and at the prompt type in cd Downloads and press enter. Then type in openssl sha1 your_filename.here and press enter. The result will the be displayed. Caution must be exercised when using the Terminal on a Mac. When you are done type exit and press enter, you can now close the Terminal application.

UNIX-based systems will vary but you should be able to use the SHA1 command from the prompt to generate the checksum.

About Password Generator Website

With more and more online accounts available, and more and more individuals looking to exploit online accounts it is vital that your passwords are different for each account you have, and are changed regularly. This site provides a way of generating random passwords to suit this need. The code behind the site creates passwords to your requirements using a cryptographically secure random number generator assigning the values to letters, numbers, or symbols as appropriate. For a secure random password make a long password with letters, numbers, and symbols. If you don't have access to a password store application this site can also generate passwords made up of two or more randomly picked words and numbers.

If you are in the business of uploading files to the web for users to download at a later date and need to provide those users reassurance, or regularly download files that come with checksums, then that is where our file integrity section comes into play. Knowing that the file you have downloaded is the genuine version, unaltered, is vital to prevent your systems from being exploited by malicious software. We don't currently have the capability to generate checksums on this site, but have provided instructions for you to generate them yourself offline, on both Windows and MacOS.

No password generated by the site is stored anywhere on our servers, and this site uses HTTPS to protect the transmission of the password to you after generation.
